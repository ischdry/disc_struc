General Function

B can have many A, meaning more than one “A” can map to the same “B”.

A function is a relation between two sets that one of the two sets has every one of its elements associated with exactly one element in the other set. A function can be one to one or many to one but not one to many.

For a function f: X → Y, (reads as “if f is a function from the set X to the set Y) then: f (x) is defined for each xX, meaning every x X participates in the function f. Each x X is associated with exactly one y Y ‘x’ is called pre-image and ‘y’ is called image of function f.
